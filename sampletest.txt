two way binding
Combination of two bindings [] -> property binding  to send property of class to html element
() -> event binding to send user data back to property of class
<input [(ngModel)]
component lifecycle hooks
OnInit -> component intialization, retrieve data
OnChanges -> perform action after change to input propertis
OnDestroy -> perform cleanup
Nested components
Components can be pass data to another components throunh @Input() and @Output() decorator
@Input() -- parent compnent pass data to child compnoent
Ex: create component and declare property with @input() rating: number
and in parent component template bind the property to component as below
<pm-star [rating]='movie.movieRating'></pm-star>
movie rating from json will be assigned to rating..
